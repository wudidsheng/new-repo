# AI生成Logo
 根据你的喜好在线生成Logo。
## 技术
`nextjs` `Tailwind Css` `Gemini API` `Clerk` `mongodb`

## 说明
 在线生成图片。根据用户的输入生成提示词语。生成符合用户预期的图片。    
用户可以免费使用5次`gemini`生成图片。可以免费使用 `deerapi` 生成两次图片
